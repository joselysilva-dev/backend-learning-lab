# MinhaPrimeiraApi

Primeira API desenvolvida em ASP.NET Core durante meus estudos de Backend.

## Tecnologias

- C#
- ASP.NET Core
- Swagger/OpenAPI
- Entity Framework Core
- Git e GitHub

## Funcionalidades

- GET /alunos
- GET /alunos/{id}
- POST /alunos

## Objetivo

Projeto criado para aprendizado de desenvolvimento Backend no ecossistema Microsoft.
